# python-lab2-eazy1-
# Отримання курсу валют НБУ за допомогою Postman

## Мета роботи
Отримати офіційний курс валют Національного банку України
за попередній тиждень з використанням REST API та Postman.

## Використані інструменти
- Postman
- REST API НБУ
- GitHub

## Опис виконання
1. Створено GET-запит до API НБУ
2. Вказано період за попередній тиждень
3. Отримано відповідь у форматі JSON
4. Результати збережено у файл та додано до репозиторію

## Результати
- response.json — відповідь API
- postman_request.png — налаштування запиту
- postman_response.png — результат виконання
